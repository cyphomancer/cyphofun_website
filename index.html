5. Launching a Solana token via Cyphofun
Prerequisites:
-Ensure you have an active Telegram account.
-Solana wallet with at least 0.3 SOL
-Create a new Telegram chat group and get the weblink as described in Step 1 below.
-The project name, ticker, project description, token image (PNG or JPG less than 4MB). Project website and Twitter accounts are optional.

Step-by-Step Instructions:

  • 1. Create a New Telegram Private Chat group
    • Click on the Create Group icon.
    • Select ‘New Group’.
    • Hit the next screen arrow.
    • Give your group a name and select the checkmark.
    • Go to the edit group option for this new chat group and select ‘Group Type’ and choose Public Channel. Then create a unique group name that will be your Telegram name for your project chat group as indicated for the Public link field. Copy and save this link. You will use it for the token launch (Example: t.me/myproject).
  • 2. Start Interaction with the Cyphofun Bot
    • Click on this link or open Telegram and search for the Cyphofun_Bot.
    • Start a conversation by clicking “Start” or sending the /start command.
    • The bot will display available commands and provide a link for detailed instructions.
  • 3. Launch a Token
    • Hit the /launch command.
    • Read and accept terms and conditions or exit the app.
    • Follow the prompts to enter each piece of information requested. The bot will also request the Telegram weblink/url that you obtained in Step 1 above.
    • You will send exactly 0.3 SOL to the wallet indicated by the bot.
    • Once the bot detects this transaction, the bot will create the token via pump.fun and provide the solscan transaction confirmation link.
    • Approximately 60 seconds after token creation, the bot will send a batch of purchased tokens to another wallet. The tokens are sent to another wallet to prevent “Dev Sold” warnings later on.
    • This wallet will hold the tokens until the market cap of the token is $250k. When this market cap is reached, the bot will sell the tokens and activate both bots for use by the user that launched the token.
  • 4. Setting Up Your Telegram Chat Group
    • Select ‘Add Admin’.
    • Hit the magnifying glass icon and enter safeguard and then OK.
    • Then DM the bot named @safeguard.
    • Click ‘Create Portal’ then ‘Create a New Group For This’. Then name the group whatever you want but it is typically your project name.
    • Click ‘Select a channel’ then select the channel that you created in Step 1.
    • Now you have an entry portal to your chat group which prevents bot spam.
    • Go to your newly created chat group (not the newly created channel) and add the two Cyphofun/Cyphomancer bots as described below.
    • Go to the edit group options.
    • Select ‘Add Admin’.
    • Hit the magnifying glass icon and enter Super_Root_Bot and then OK.
    • Most projects want this chat group to be available to the public so you won’t use the token-gate bot here.
    • Go ahead and setup Root (Super_Root_Bot) for this group by following the instructions when you direct message (DM) Root. They can also be found here.
  • 5. Setting Up Your Telegram Whale Holder Chat Group
    • If you wish to create a separate whale holder chat group for users that hold above a certain amount of your token then follow these instructions.
    • Click on the Create Group icon.
    • Select ‘New Group’.
    • Hit the next screen arrow.
    • Give your group a name and select the checkmark.
    • Go to the edit group option for this new chat group and select ‘Group Type’ and make sure that ‘Private Group’ is selected. Copy and save the Invite link. You will use it for setting up the token-gate and also share it with your community so that they can try and join your token-gated group.
    • Select ‘Add Admin’.
    • Hit the magnifying glass icon and enter Cyphomancer_Gate_Bot and then OK.
    • Now follow the detailed instructions for setting up the token-gated group here.
    • You can also find the instructions for your community to access your new whale holder group here.
